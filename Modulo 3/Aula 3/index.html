Com a propriedade display: inline;, os elementos e textos HTML ficam posicionados lado a lado, sem quebra de linha entre eles. No exemplo acima, quando o símbolo || aparece, normalmente haveria uma quebra de linha. block

Display block no span Display block no span Display block no span
Normalmente, o elemento span não provoca quebras de linha, pois seu comportamento padrão é o de um elemento inline. No entanto, ao aplicar a propriedade display: block;, ele passa a se comportar como um elemento de bloco, ocupando toda a largura disponível e forçando uma quebra de linha antes e depois. Isso altera sua apresentação, mesmo que esteja dentro de um único parágrafo.

flex

Com a propriedade display: flex;, os elementos são organizados de forma adjacente, ou seja, lado a lado, sem quebra de linha automática como ocorre com elementos em bloco. Diferente do comportamento padrão dos elementos block, que ocupam toda a largura e forçam uma nova linha, o flex permite que os itens fiquem alinhados horizontalmente (ou verticalmente, dependendo da direção definida), mantendo o controle total sobre tamanho, espaçamento, alinhamento e distribuição.

Quando utilizamos a propriedade z-index, os elementos com valores mais altos têm prioridade na ordem de empilhamento, aparecendo acima dos demais. Por exemplo, mesmo que o div.area-1 seja o primeiro elemento na hierarquia do HTML, ele pode aparecer por cima dos outros se tiver um z-index maior, como z-index: 5, o que lhe confere maior destaque visual.
