display: grid;

grid-template-columns:repeat(3,1fr);
grid-template-rows: repeat(4,1fr);
grid-template-areas:
"header header header"
"nav nav nav"
"content content content"
"footer footer footer"
grid-template-areas Define áreas nomeadas para facilitar o posicionamento dos elementos filhos. Cada linha tem 3 colunas, mas todas as áreas (header, nav, content, footer) estão ocupando todas as 3 colunas ou seja, são áreas que se estendem horizontalmente. Então no caso da letra A, o div que possuia a o grid-area com o termo header, vai se estender pelos 3 espaços que ele ocupa.
