JavaScript

Clique com o botão direito do mouse e selecione Inspecionar. Ao abrir, redirecione para a aba console, assim poderá ver o que está sendo feito em js.
var
Como o nome já sugere, ele é uma variável, mas seu diferencial é que ele é acessível de qualquer lugar dentro do escopo.
let
É uma variável que só é acessível onde ela foi declarada, então se em um if for utilizado o a variável let, seu conteúdo só será acessível dentro desse if.
const
Como o nome já sugere, ele é uma variável constante que não pode ser alterada, se ela foi declarada como um int de valor ‘1’ ela será isso para sempre.
string
Texto entre aspas simples ou duplas simbolizando o que é texto
number
Números inteiros ou decimais
boolean
Verdadeiro ou falso
null
Ausência intencional de valor
bigint
Números inteiros muito grandes
symbol
Identificadores únicos e imutáveis - Symbol("id")
