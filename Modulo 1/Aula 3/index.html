A classe em HTML é um atributo usado para agrupar elementos que compartilham o mesmo estilo ou comportamento. Ela permite aplicar regras CSS ou manipulações JavaScript a vários elementos ao mesmo tempo. Basta usar class="nome-da-classe" para identificar os elementos que pertencem àquela categoria.
